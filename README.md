# High-Low Dice Duel

Simulates rolling two six-sided dice. The user guesses Low (sum 2-6), Mid (sum 7), or High (sum 8-12). The program displays the roll, determines the round winner, updates running scores, and repeats until the user chooses to quit.

Results: Displays each roll and round outcome, running score after each round, and final aggregate results when the session ends.
